# PRD: First Meet Git

## 1. 产品概述
面向 JavaScript 开发初学者的 Git 入门教学项目，旨在帮助学员掌握版本控制基础操作。

## 2. 目标用户
- JavaScript 开发初学者
- 无 Git 使用经验的学员

## 3. 核心功能

### 3.1 基础操作
- 初始化 Git 仓库 (`git init`)
- 添加文件到暂存区 (`git add`)
- 提交更改 (`git commit`)
- 查看仓库状态 (`git status`)
- 查看提交历史 (`git log`)

### 3.2 分支管理
- 创建与切换分支 (`git branch`, `git checkout` / `git switch`)
- 合并分支 (`git merge`)
- 解决合并冲突

### 3.3 远程仓库
- 关联远程仓库 (`git remote add`)
- 推送与拉取 (`git push`, `git pull`)
- 克隆仓库 (`git clone`)

### 3.4 进阶操作
- 撤销更改 (`git reset`, `git revert`)
- 暂存工作区 (`git stash`)
- 查看文件差异 (`git diff`)

## 4. 项目结构
```
first-meet-git/
├── README.md          # 项目说明
├── PRD.md             # 产品需求文档
└── exercises/         # 练习题目
```

## 5. 成功指标
- 学员能独立完成 Git 常用操作
- 学员理解版本控制的核心概念（提交、分支、合并）
- 学员能使用 Git 进行基本的团队协作

## 6. 时间规划
- Week 1: Git 基础概念与本地操作
